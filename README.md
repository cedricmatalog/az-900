# AZ-900: Microsoft Azure Fundamentals - Complete Study Guide

**Everything you need to pass the AZ-900 certification exam**

## About This Guide

This comprehensive study guide is designed to prepare you fully for the Microsoft Azure Fundamentals (AZ-900) certification exam. After reading and understanding this material, you will have the knowledge needed to pass the exam.

## Exam Information

- **Exam Code**: AZ-900
- **Duration**: 60 minutes
- **Number of Questions**: 40-60 questions
- **Question Types**: Multiple choice, multiple select, drag-and-drop, scenario-based
- **Passing Score**: 700 out of 1000 points
- **Cost**: $99 USD
- **Languages**: Available in multiple languages
- **Prerequisites**: None (entry-level certification)

## Official Skills Measured

Based on the official Microsoft exam objectives:

1. **Describe Cloud Concepts** (25-30%)
2. **Describe Azure Architecture and Services** (35-40%)
3. **Describe Azure Management and Governance** (30-35%)

## Exam Tips and Strategies

1. **Read Questions Carefully**: Look for keywords like "least cost", "highest availability", "minimum administrative effort"
2. **Use Elimination**: Cross out obviously wrong answers first
3. **Time Management**: You have about 1-1.5 minutes per question
4. **Don't Overthink**: This is a fundamentals exam - answers are straightforward
5. **Scenario Questions**: Focus on understanding the requirement before choosing
6. **Mark for Review**: Flag uncertain questions and return to them
7. **No Negative Marking**: Always provide an answer, even if guessing
